College Event Feedback Analysis
       This project analyzes student feedback from college events using Python to understand satisfaction levels through sentiment analysis.

# Tools Used
- Google Forms  
- Google Colab  
- Pandas, TextBlob, Matplotlib  

# Steps
1. Collected feedback using Google Forms  
2. Downloaded data as CSV  
3. Cleaned and analyzed data in Colab  
4. Performed sentiment analysis  
5. Created a bar chart visualization  

# Result
Feedback was classified into Positive, Neutral, and Negative sentiments.
